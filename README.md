# oreoluwa-portfolio                                                                                                                                                                                                                                                                                                                         # Portfolio: Trust Architecture in Fintech

A professional portfolio focused on product strategy, user research, and trust-centered design within the African fintech ecosystem.

## 🚀 The Vision
Building digital products is easy; building digital products people *trust* is the challenge. This portfolio showcases my journey diagnosing friction in onboarding funnels and designing "Trust Architecture" to improve user activation.

## 🛠️ Tech Stack
- **Frontend:** Tailwind CSS (for rapid, responsive UI development)
- **Icons:** Lucide-React
- **Architecture:** Clean, semantic HTML5
- **Documentation:** Notion (linked case studies)

## 📂 Key Case Studies Included
- **Risevest:** Diagnosing activation barriers in investment onboarding (Email -> BVN -> ID).
- **OnePipe:** Solving the "Ghost Debit Panic" through a Cortex Engine architecture.
- **Bamboo:** UX audit on false completion patterns in KYC.
- **BeReal:** Growth strategy via user-controlled "Streak Circles."

## 📈 Impact & Research
This site isn't just a resume; it's a collection of data-driven insights gathered from over **80+ structured user conversations** and deep-dive UX audits.
